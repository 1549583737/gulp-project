# gulp-project

> 一个基于gulp构建的项目

## 项目初始化步骤

#### 1、安装nodejs环境

#### 2、全局安装gulp
``` bash
    命令: (sudo) npm install -g gulp
```

#### 3、在项目根目录执行npm初始化
``` bash
    命令: npm install
```

#### 4、启动项目
``` bash
    开发模式: npm run dev
    生产模式: npm run build
```
